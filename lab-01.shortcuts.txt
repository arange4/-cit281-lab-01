Copy selected text to clipboard (copy/paste buffer) - ctrl-shift-c/command-c
Paste from buffer- ctrl-shift-v/command-v
Cut selected text to buffer (selected text is deleted)- command-x
Save current file in VSCode- command-s
Toggle block comment in VSCode- shift-option-a
Toggle line comment in VSCode- shift-option-a
Move cursor from word to word right- ctrl-right arrow
Move cursor from word to word left- ctrl-left arrow
Select word right- hold down ctrl + shift, press right arrow key
Select word left- hold down ctrl + shift, press left arrow key
Select all contents- ctrl-a
Select current line, including end of line character- hold shift and press end 
Jump to start of line- shift + enter
Jump to end of line- ctrl + home
Jump to beginning of current file- Key in 1 + G
